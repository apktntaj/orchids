Analisis Jumlah Cases (Box) per Pelanggan

Berdasarkan konversi yang diberikan, kami menghitung total cases per pelanggan dengan rumus: cases = pallets * konversi box_size.

In[5]:
# Konversi box_size ke cases
conversions = {
    '11lb': 200,
    '9lb': 220,
    '7lb': 260,
    '4lb': 448,
    '2lb': 1024,  # 64mc = 1024cs
    '1lb': 1536,  # 64mc = 1536cs
    '25lb': 96
}

# Hitung cases per baris
df['cases_calculated'] = df.apply(lambda row: row['pallets'] * conversions.get(row['box_size'], 0) if pd.notna(row['pallets']) else 0, axis=1)

# Total cases per pelanggan
total_cases_per_customer = df.groupby('customer_name')['cases_calculated'].sum().sort_values(ascending=False)
print("Total cases per pelanggan:")
print(total_cases_per_customer)

# Plot
total_cases_per_customer.plot(kind='bar', figsize=(10, 6))
plt.title('Total Cases per Pelanggan')
plt.xlabel('Pelanggan')
plt.ylabel('Total Cases')
plt.xticks(rotation=45)
plt.show()
